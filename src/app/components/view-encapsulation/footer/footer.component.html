ViewEncapsulation.None

None — Styles Go Global

Styles are NOT scoped — they leak into the whole page!

Angular writes the component CSS directly to the <head> as a plain stylesheet. Any class name defined here becomes a global style.

  • ⚠️ Risk — can unintentionally override sibling styles
  • ✅ Useful for theming/overlay libraries
  • ✅ Easiest to override from outside
👆 The .leak-demo red border style you see here is defined in this component's CSS — but it will also affect any other element with class leak-demo anywhere in the app!
